# LAQSHYA BADMINTON ACADEMY - Quick Start Guide

## üöÄ Getting Started in 3 Steps

### Step 1: Create Your Admin Account
1. Open the application
2. Click "Sign In" in the header
3. Go to "Register" tab
4. Fill in your details:
   - Full Name
   - Username (will be your admin username)
   - Password
5. Click "Create Account"
6. **You are now the admin!** (First user automatically becomes admin)

### Step 2: Explore the Platform

#### As a User:
- **Home Page**: View sports offerings, benefits, and amenities
- **Book Now**: Make your first booking
  1. Select a sport (Badminton/Cricket/Tennis)
  2. Choose venue, ground, date, and time
  3. Apply a discount coupon (try: WELCOME50)
  4. Confirm booking
- **My Dashboard**: View your bookings and manage profile

#### As an Admin:
- Click "Admin" button in header
- **Admin Dashboard** gives you access to:
  - User management
  - Booking overview
  - Coupon management
  - Revenue statistics

### Step 3: Test Key Features

#### Test Booking Flow:
```
1. Go to "Book Now"
2. Select "Badminton"
3. Choose "LAQSHYA Main Arena"
4. Select "Badminton Court 1"
5. Pick today's date
6. Choose an available time slot (green)
7. Apply coupon "WELCOME50"
8. Confirm booking
9. Check "My Dashboard" to see your booking
```

#### Test Admin Features:
```
1. Click "Admin" in header
2. Go to "Coupons" tab
3. Click "Create Coupon"
4. Create a test coupon:
   - Code: TEST10
   - Type: Percentage
   - Value: 10
   - Min Amount: 100
5. Save and activate
6. Try using it in a booking
```

## üêõ Troubleshooting

### Can't Login?
- Make sure you're using the correct username format
- Username is automatically appended with @miaoda.com

### Booking Not Showing?
- Check "My Dashboard" for booking history
- Ensure payment was completed
- Refresh the page

### Coupon Not Working?
- Check minimum amount requirement
- Verify coupon is active
- Check expiry date
- Ensure usage limit not reached

### Admin Features Not Visible?
- Confirm you're the first registered user
- Check your role in the database
- Try logging out and back in
